# HyperDrift
**Drift.Loop.Smash.Repeat**

Made for the GMTK Game Jam 2025, in a team of two!

### Overview
Pilot a rogue car built to smash through waves of robot attacks. Drift in sharp loops to turn the robots into scrap. Master precision drifting, and activate Attack Mode to unleash chaos on the battleground!
